# SSD - Spatial Scene Dataset 

Includes the following:
- scripts for autolabelling videos using Gemini (/label)

### Data

- Pexels (copyright free videos): https://www.pexels.com/search/videos/objects/
- Video Object Relation Dataset (from Yahoo): ttps://xdshang.github.io/docs/vidor.html
- Multiple Object Tracking Dataset https://motchallenge.net/data/MOT17/
- Recognition of Human Actions (mostly for movement like walking or running in different directions): https://www.csc.kth.se/cvap/actions/
- UCF101 Action Recognition Dataset: https://www.crcv.ucf.edu/data/UCF101.php
